📂 Installation Steps

  1. Download all three icons above
  2. Move them to extension/icons/ as icon16.png, icon48.png, icon128.png
  3. Open Chrome → chrome://extensions
  4. Enable Developer mode (top-right toggle)
  5. Click Load unpacked → select the extension/ folder
  6. The AgentShield icon appears in your toolbar — click it to open the mini-dashboard!
